ねこ学びログ Lite｜静灯工房
猫と暮らす人のための
静かでシンプルな学びの記録アプリです。

動画・記事・本などのURLから
タイトルやタグを自動推定し、
日々の学びを落ち着いて整理できるように設計しています。

📘 このアプリについて
「猫のために学んだこと」を
行動ベースで記録するための軽量トラッカーです。

URLからタイトル・タグを推定
進捗（ページ数・動画時間など）を管理
簡易メモで気づきを残せる
今日の学びを「足跡」として可視化
全体の積み重ねを「知識の木」で表現

すべてブラウザ内で完結し、
データは端末のローカルに保存されます。

🌱 主な機能
🐾 学びの追加
URL・タイトル・シリーズ・タグ・総量・現在位置・メモを登録できます。
タグは URL から自動推定され、手動で調整も可能です。

🌳 ねこの知識の木
学びの積み重ねを、淡色のミニマルな木として表示します。

👣 今日の足跡
その日に追加した学びを、
シンプルな足跡アイコンで確認できます。

📄 学び一覧
進捗バー・タグ・シリーズで整理された一覧。
検索・並び替えにも対応しています。

💾 ローカル保存
データはすべてlocalStorageに保存され、
バックアップ（JSON）として出力・復元できます。

🌿 コンセプト
淡色・静けさ・余白を大切にした
ミニマルな学習ツールです。

1ページで完結する軽量設計
スマホでも読みやすいUI
実用性と優しさを両立した学びの整理
猫のための学びを「行動」として蓄積する設計

📱 インストール方法（PWA）
ねこ学びログ Liteは
ブラウザからそのまま “アプリ化” できます。

✔ スマホ（iPhone / Android）
公開ページを開く
ブラウザの共有メニュー（または設定）を開く
「ホーム画面に追加」 を選ぶ

アイコンがホーム画面に追加され、
普通のアプリのように起動できます

✔ PC（Chrome / Edge）
公開ページを開く
アドレスバー右端の インストールアイコン（＋） を押す
「インストール」を選択

デスクトップにアプリとして追加されます

※ インストール後は オフラインでも利用可能です。

💾 バックアップ仕様
データはすべてブラウザの
localStorage（端末内）に保存されます。

✔ バックアップ保存（JSON）
「バックアップを保存」ボタンを押すと
現在の学びデータをJSONファイルとして出力します

ファイル名例：
neko-manabi-backup-2026-07-14.json

✔ バックアップ復元
JSONファイルを選択すると
保存時点のデータがそのまま復元されます

復元後は自動で一覧が再描画されます

✔ 注意点
データは端末ごとに保存されるため、
別の端末ではバックアップ復元が必要です

Chrome同期がオンの場合は
同期された端末間で自動引き継ぎされます

🛠 技術構成
HTML / CSS / JavaScript の静的サイト
ブラウザのlocalStorageを利用したローカル保存
PWA（manifest.json）対応でホーム画面に追加が可能

📁 ファイル構成（例）
index.html … 画面構成
style.css … レイアウト・デザイン
script.js … ロジック・ローカルストレージ処理
manifest.json … PWA 用設定
icons/ … アイコン画像など

🔒 データと安全性
学びの内容・進捗・メモは すべて端末内に保存されます
開発者や第三者のサーバーには送信されません
バックアップJSONもユーザー自身の端末に保存されます
